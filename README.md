## Requirements

* PHP 8.0 or latest
* Node 16+ or latest

## How to run

```bash
$ cp .env.example .env # configure app for laravel
$ touch database/database.sqlite
$ composer install
$ npm install
$ npm run dev # compiling asset for development
```
